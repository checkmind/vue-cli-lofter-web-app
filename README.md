# vue-cli-lofter-web-app
模仿lofter app的h5 app
## 一 为什么要模仿lofter  
因为本人觉得lofter的界面十分优美，并且规模也不小。  
并且刚写了vue的入门项目，因此需要实践项目巩固一下知识。

### 安装步骤  
安装依赖: npm install  

安装vue-cli: npm install --global vue-cli  

运行: npm run dev
